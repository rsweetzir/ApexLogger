What it does

Everything at a glance

🏁

GPS Lap & Run Timing

Set a virtual start/finish line anywhere. ApexLogger auto-triggers on crossing and records split times with millisecond precision. Works without OBD.

📡

OBD-II Telemetry

Connect a Bluetooth OBD-II interface to stream real-time boost, throttle, oil temp, coolant, battery voltage, and engine-specific data.

⛽

Fuel Pressure Monitoring

HPFP and LPFP live monitoring for N54/N55/B58, plus fuel rail pressure for RS3, GTI, Golf R, and XC60 — with configurable warning thresholds.

🔧

Per-Cylinder Timing

See timing corrections for all cylinders in real time. Catch knock before it becomes a problem.

↕ ↔

G-Force Display

Four-directional G-force arrows surround the speedometer — visible even in GPS-only mode without any OBD connection. Powered by the device IMU.

🗺

Walk the Course

Walk your layout on foot before driving. ApexLogger maps your GPS path, marks waypoints for cones and gates, and stores it for reference during the session.

📂

GPS Course Import

Load a .gpx or .kml file from your event organizer. ApexLogger auto-detects Start, Finish, and waypoints by name — no manual placement needed.

⌚

Apple Watch Heart Rate

Arm a workout session from your wrist or phone. Live BPM appears on the gauges page and is recorded with every track point.

📊

Session Review

Browse saved runs with peak speed, elapsed time, and point count. Up to five sessions stored locally.

🚗

OBD-Free Mode

No interface? No problem. GPS speed with G-force arrows and MPH / KPH toggle is always available — no OBD connection required.

Event modes

Built for every format

Autocross

Cone Course

Independent Start and Finish lines set by tapping or walking the course. Add waypoints for cones and gates. Walk mode maps the layout on foot before you drive.

Drag

Straight-Line

Cross the GPS start line or ARM for auto-Go on first throttle. Finish is auto-triggered at your chosen distance — ⅛ mile, ¼ mile, or a custom distance.

Track Day

Circuit

A single combined Start/Finish line for lap timing. Walk or run a recon lap to learn the track. Timer triggers on every crossing of the line.

Rally

Stage Rally

ARM → auto-Go on first motion. Each stage ends automatically at the selected distance. Stage counter increments between runs. Unlimited stages per event.

Switch event mode from the Track screen or directly from the CarPlay driver panel — no phone interaction required once you're strapped in.

Apple CarPlay

Gauges on the dash.
Controls at your fingertips.

ApexLogger fills the CarPlay screen with live telemetry — no map, no distractions. A large driver-side action panel gives you one-tap control of everything you need without touching your phone.

  • ARM / STOP timer with auto-launch on throttle input
  • Set Start, Finish, and waypoints per event type
  • Race mode selector: Autocross, Drag, Track Day, Rally
  • Drag distance toggle: ⅛ mi ↔ ¼ mi with a single tap
  • Rally stage counter with long-press reset
  • LHD / RHD layout swap for driver-side preference
  • Phone automatically shows the track map while CarPlay is active
